# Swiggy SQL Analysis

This project simulates a food delivery platform database and performs analytical SQL queries to derive business insights.

## Dataset
The project contains tables representing:

- Customers
- Restaurants
- Delivery Agents
- Orders
- Order Items

## SQL Skills Demonstrated

- Joins (INNER JOIN, LEFT JOIN)
- Aggregations (SUM, COUNT, AVG)
- Window Functions (ROW_NUMBER, DENSE_RANK, Running Totals)
- Common Table Expressions (CTE)
- Views
- Subqueries

## Analysis Performed

- Restaurant revenue analysis
- Top spending customers
- Popular food items
- Restaurant performance ranking
- Running revenue trends
- Customer ordering behavior
- Consecutive day ordering detection

## Files

01_schema.sql → Database and table creation  
02_data.sql → Sample dataset insertion  
03_analysis.sql → Analytical SQL queries
