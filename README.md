This repository contains SQL-based data analysis projects focused on solving real-world business problems using structured query language (SQL).

# Projects Included

## 1. E-Commerce Analysis

This project focuses on analyzing e-commerce business data to extract meaningful insights related to customers, orders, products, sales, and revenue.

### Key Analysis Performed
- Customer purchase behavior
- Revenue analysis
- Top-selling products
- Order trends
- Category-wise performance
- Customer segmentation
- Sales insights using SQL queries

### SQL Concepts Used
- SELECT Statements
- WHERE & HAVING Clauses
- GROUP BY
- ORDER BY
- JOINS
- Subqueries
- Aggregate Functions
- Window Functions

---

## 2. Swiggy Data Analysis

This project analyzes food delivery data inspired by Swiggy to understand restaurant performance, customer preferences, and delivery trends.

### Key Analysis Performed
- Restaurant ratings analysis
- Most popular cuisines
- Price comparisons
- Delivery performance insights
- Customer ordering patterns
- City-wise restaurant analysis

### SQL Concepts Used
- Joins
- Nested Queries
- Ranking Functions
- CASE Statements
- Aggregate Functions
- Filtering & Sorting
